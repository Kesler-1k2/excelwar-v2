# Excel Wars
ExcelWars is designed to be a fun and helpful learning buddy for Computing students to learn Excel from scratch. Excel is a skill that is extremely useful in the workplace. However, many still find it hard to learn. ExcelWars aims to change that. With a web spreadsheet softare that can run just like Excel, and an AI buddy to help in lessons, anyone can learn Excel on the go. ExcelWars can also benefit people with time constraints as it consolidates all the useful and necessary information on Excel to make their learning experience smoother and more efficient.
